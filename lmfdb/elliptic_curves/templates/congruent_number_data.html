Data for the first 1,000,000 Congruent Number Elliptic Curves is available, provided by Randall L. Rathbun. For each of the first 1,000,000 positive integers $n$, this includes data for the $n$th congruent number elliptic curve $E_n$, from which it can be determined whether or not $n$ is a congruent number, and a solution to the congruent number problem when it is.

The congruent number problems, congruent numbers and congruent number curves

A congruent number $n$ is a rational number which is the area $\frac{1}{2}XY$ of a right triangle with rational legs $X$, $Y$ and rational hypotenuse $Z$, so that \[ X^2 + Y^2 = Z^2 \tag{1} \] and \[ \frac{1}{2} XY = n. \tag{2} \] The congruent number problem is the problem of determining whether a given rational number is or is not a congruent number. Since $n$ is congruent if and only if $nt^2$ is, for any nonzero rational number $t$, it is sufficient to consider the case where $n$ is a square-free integer; nevertheless, the data here includes the 392074 non-squarefree integers up to one million as well as the 607926 squarefree integers.

For example, $6$ is a congruent number because it is the area of the $3,4,5$ right triangle.

A positive integer $n$ is congruent if and only if there exists a rational number $a$, such that $a$, $a-n$ and $a+n$ are all squares of rational numbers. For, given $X,Y,Z$ satisfying (1) and (2), $a=(Z/2)^2$ satisfies $a\pm n=((X\pm Y)/2)^2$, and conversely, given $a=A^2$ with $a+n=B^2$ and $a-n=C^2$, set $(X,Y,Z)=(2A,B+C,B-C)$.

For example, with the $3,4,5$ right triangle and $n = 6$, let $a = 25/4=(5/2)^2$; then $a-n=25/4 - 6 = (1/2)^2$ and $a+n=25/4 + 6 = (7/2)^2$.

It is known that a positive integer number $n$ is congruent if and only if there exists a rational solution $(x,y)$ with $y\not=0$ to the {{ KNOWL('ec', title='elliptic curve') }} equation: \[ E_n:\quad y^2 = x^3 - n^2 x, \] or, in other words, a rational point $P=(x,y)\in E_n(\Q)$ with $y\not=0$. (Given $(X,Y,Z)$, set $x=\frac{1}{4}Z^2$ and $y=\frac{1}{8}Z(X^2-Y^2)$.) Moreover, the only rational points of finite order on $E_n(\Q)$ are the point at infinity (the origin for the group law) and the three points of order $2$: $(0,0)$, $(\pm n,0)$; thus, $n$ is congruent if and only if $E_n(\Q)$ is infinite, that is, if and only if the {{ KNOWL('ec.rank', title='rank') }} of $E_n(\Q)$ is positive. For example, the elliptic curve $E_6: y^2 = x^3 - 36x$ has rank $1$, with Mordell-Weil group $E_6(\Q)$ generated (modulo torsion) by $P = (-3,9)$, and $2P=(25/4,-35/8)$ is the rational point arising from the $(3,4,5)$-triangle. In general, only points in $2E_n(\Q)$ give solutions to the congruent number problem.

Note that that each congruent number elliptic curve $E_n$ belongs to an {{ KNOWL('ec.isogeny_class', title='isogeny class') }} of 4 isogenous curves, connected by {{ KNOWL('ec.isogeny', title='2-isogenies') }}. The curves $2$-isogenous to $E_n=[0,0,0,-n^2,0]$ are $[0,0,0,4n^2,0]$ and $[0,\pm 12n,0,n^2,0]$. When finding generators for each curve, it was sometimes easier to first find generators for the isogenous curves, when these had smaller height.

{{ KNOWL('lfunction.sign', title='Root number') }}

The {{ KNOWL('lfunction.sign', title='root number') }} of an elliptic curve is the sign ($+1$ or $-1$) of the {{ KNOWL('lfunction.functional_equation', title = 'functional equation') }} satisfied by its {{ KNOWL('lfunction', title = 'L-function') }}. It is also equal to $(-1)^{r_{\text{an}}}$ where $r_{\text{an}}$ is the {{ KNOWL('ec.q.analytic_rank', title='analytic rank') }} of the elliptic curve, and hence by the {{ KNOWL('ec.q.bsdconjecture', title='Birch-Swinnerton-Dyer Conjecture') }} is expected to be equal to $(-1)^r$ where $r$ is the rank of the {{ KNOWL('ec.mordell_weil_group', title='Mordell-Weil group') }}.
